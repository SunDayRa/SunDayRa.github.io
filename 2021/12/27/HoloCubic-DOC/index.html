整体思路

本文涉及到的软件在交流群与NAS个人盘(持续补充中……)中均有提供

一、前期准备

1. Altium Designer安装

  • 具体安装及POjie方法请百度（防止域名被和谐）

2. VSCode&VS2019

只为模拟环境，建议使用VS2019，不用担心网络问题

修改程序建议使用VSCode

1. 下载安装

  1. 个人网盘VSCode 官方网址，选择System Installer 64bit 一直下一步下一步即可
  2. Visual Studio 2019
    • 下载社区版，下载完成得到一个文件vs_Community.exe
    • 双击打开
    • 环境基于C语言 选择C++ 安装即可
    • 此时模拟仿真Lvgl的朋友即可打开.sh 文件进行模拟

下面全部基于VSCode安装

2. Platformio IDE 插件安装

  • 点击安装即可，需要等待较长时间(国外服务器)

3. MSYS2

下载MSYS2，建议安装到默认位置

Win搜索框搜索MSYS2打开

执行命令，按Shift+Ins粘贴（或者右键，Paste），然后回车。提示输入y即可

添加环境变量

将 Mingw-w64bin文件夹的路径添加到 Windows PATH 环境变量（通常为C:\msys64\mingw64\bin）。

Win搜索框搜索：编辑系统环境变量通常为C:\msys64\mingw64\binps:更改默认路径的去对应目录找。

然后一直确定。

2. Fusion 360

下载地址：官方网址 使用30天，有学生邮箱的适用1年。或者淘宝买学生账号(网购有风险，下单需谨慎哦！)

安装一直下一步就行。

3. HoloCubic源码文件

去作者Github仓库下载即可，或者NAS个人盘

好，准备工作到此结束。
